このサイトについて

  • 概要
  • 小児歯科の架空サイトです。
  • 制作期間
  • 10日
  • ターゲット
  • 3～12歳の子どもを持つ親（クリニックの近隣に幼稚園と小学校があると仮定）
  • 配色
  • 子どもが安心できる空間をイメージして、パステルカラーで配色
  • 使用ツール
  • Visual Studio Code/Photoshop
  • 使用言語
  • HTML/CSS/JavaScript

工夫した点

  • ターゲットがデジタルネイティブ世代であることからスマートフォンで検索されることが多いと想定し、 モバイルファーストでサイト全体の構成を考えました。 PC用に向けたレスポンシブデザインも実装しています。
  • スマートフォンでサイトを見たときのメニューの見やすさを意識し、 Java Scriptでスライドメニューを実装しました。 PC版ではメニューを画面右側に固定しています。
  • 予約用のフォームを実装しました。 すぐに予約ページに遷移できるよう、リンクボタンをトップページのファーストビューエリアに配置しています。
